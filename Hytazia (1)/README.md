# Hytazia - Site & Boutique pour serveur Minecraft PvP Faction

Contenu : frontend (Next.js skeleton), backend (Express/TypeScript skeleton), SQL schema, Stripe + Microsoft auth helpers, Socket/Redis notes, Spigot plugin skeleton, Docker Compose for local testing, images (logo & presentation).

Ce repo est un **kit** prêt à être développé et déployé. Les scripts et fichiers fournis sont des squelettes. Remplace les valeurs d'environnement avant d'utiliser en production.
